# Headlock Jame Android

App Android WebView + tâm ảo overlay.

## Cách dùng
1. Mở thư mục này bằng Android Studio.
2. Sync Gradle.
3. Run app.
4. Bấm "Bật tâm ảo".
5. Cấp quyền "Hiển thị trên ứng dụng khác".
6. Quay lại app và bấm bật lại.

Lưu ý: một số game có thể chặn hoặc cấm overlay theo điều khoản của họ.
